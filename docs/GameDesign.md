# ⭐ سیستم‌های ویژه بازی

## 1. Team Chemistry (شیمی تیم)

شیمی تیم بر اساس:

- ملیت
- باشگاه
- لیگ
- اسطوره‌ها

### تاثیرات

+ پاس دقیق‌تر
+ جاگیری بهتر
+ افزایش احتمال گل
+ هماهنگی دفاع

اگر شیمی پایین باشد:

- پاس اشتباه
- جاگیری ضعیف
- خطای بیشتر
- کاهش عملکرد تیم
  ## 2. Manager AI (هوش مربی)

هر تیم یک مربی دارد.

انواع مربی:

- دفاعی
- هجومی
- متعادل
- ضد حمله
- پرس شدید
- مالکانه

هر مربی Level دارد.

Lv1 = 2%

Lv20 = 8%

Lv50 = 15%

تاثیرات:

- تغییر سبک بازی
- افزایش هماهنگی
- تغییر تصمیم‌های هوش مصنوعی
- افزایش احتمال برد
